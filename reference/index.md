# Package index

## Core functions

Read, annotate, and visualize BRASS fusions

- [`read_brass()`](https://bczech.github.io/BRASSVis/reference/read_brass.md)
  : Read BRASS BEDPE output
- [`load_annotations()`](https://bczech.github.io/BRASSVis/reference/load_annotations.md)
  : Load annotation files for fusion visualization
- [`plot_fusion()`](https://bczech.github.io/BRASSVis/reference/plot_fusion.md)
  : Plot a single fusion event
- [`brass_report()`](https://bczech.github.io/BRASSVis/reference/brass_report.md)
  : Generate a full BRASS fusion report
